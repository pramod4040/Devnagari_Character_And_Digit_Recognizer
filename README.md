# Nepali Character And Digit Recognization Models

This is a web-based platform that allows users to write Devanagari characters and digits and automatically predicts them.



[![Demo Video](https://img.youtube.com/vi/d-J_OkL-Ioo/0.jpg)](https://www.youtube.com/watch?v=d-J_OkL-Ioo)
